# yii2-fancybox
Yii2 FancyBox extension

Form more information on how to use FancyBox please go to their website http://fancybox.net

Example usage :
```php
    echo FancyBox::widget([
        'type' => 'image',
        'item' => [
            'href' => 'url_to_thumbnail',
            'src' => 'url_to_big_image',
        ],
    ]);
```
